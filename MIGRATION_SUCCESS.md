## 🧪 驗證步驟

### 步驟 1：測試 API 健康檢查

在瀏覽器訪問：
```
https://your-app.onrender.com/api/health
```

應該返回：
```json
{
  "status": "ok",
  "message": "Vote API is running"
}
```

### 步驟 2：測試投票列表 API

訪問：
```
https://your-app.onrender.com/api/polls
```

應該返回：
```json
{
  "polls": []
}
```

（空數組是正常的，因為還沒有投票）

### 步驟 3：測試前端

1. **訪問前端網站**
   - 前往：https://votepractice.netlify.app

2. **檢查是否正常**
   - 應該不再顯示 500 錯誤
   - 應該能看到投票列表（即使為空）
   - 應該能看到「建立投票」按鈕（需要登入）

3. **測試註冊**
   - 點擊「註冊」
   - 輸入手機號（10 碼，09 開頭）和密碼
   - 應該能成功註冊

4. **測試登入**
   - 使用剛才註冊的帳號登入
   - 應該能成功登入

5. **測試創建投票**
   - 登入後，點擊「建立投票」
   - 填寫投票信息並創建
   - 應該能成功創建

6. **測試投票**
   - 查看投票列表
   - 點擊一個投票
   - 選擇選項並投票
   - 應該能成功投票並看到結果
